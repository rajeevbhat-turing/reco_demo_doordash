# Unit Test Templates

These are template files copied from E2E fundamentals that can serve as references for creating unit tests.

## Template Files

| File | Description |
|------|-------------|
| `01-auth.spec.ts` | Authentication flow template |
| `02-address.spec.ts` | Address management template |
| `03-stores.spec.ts` | Store browsing template |
| `04-reviews.spec.ts` | Review submission template |
| `05-reorder.spec.ts` | Reorder functionality template |
| `06-checkout.spec.ts` | Checkout flow template |

## Usage

These templates show the structure and patterns used in E2E tests. 
When writing unit tests, adapt these patterns for component-level testing with Jest/Vitest.

## Note

These are E2E test templates - unit tests will use different testing libraries (Jest, Vitest, React Testing Library).
